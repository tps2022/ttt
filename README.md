# Tic Tac Toe
Tic Tac Toe game created using HTML, CSS, and Javascript
hosted on Github Pages
<hr>
About the Game: Tic Tac Toe is traditionally played on a 3 Ã— 3 grid. Players take turns placing a mark in one of the cells of the grid. The goal of the game is for players to position their marks so that they make a continuous line of three cells vertically, horizontally, or diagonally.     
<br /><br />
Features -  reset button to restart game, and a grid of 9 squares for game play, game status update
